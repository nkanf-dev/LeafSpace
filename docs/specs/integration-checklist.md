# LeafSpace 集成检查清单

## 提交前自查

### 通用
- [ ] 已阅读 `PRD.md`、`TED.md`
- [ ] 已阅读 `docs/specs/shared-contracts.md`
- [ ] 没有重复定义公共类型
- [ ] 没有无说明地修改他人 ownership 文件
- [ ] 对外接口已写清楚

### 质量
- [ ] TypeScript 类型通过
- [ ] 没有明显未处理异常
- [ ] 空态 / 加载态 / 错误态至少有基础处理
- [ ] 没有硬编码测试数据残留到正式入口

### 集成
- [ ] 与依赖模块的输入/输出对齐
- [ ] 快捷键未与全局已定义行为冲突
- [ ] store 状态不会互相覆盖
- [ ] 持久化字段可序列化

### PR 说明必须包含
- 改动文件列表
- 新增对外接口
- 依赖哪些其他 agent 的能力
- 已知限制与后续建议

# Agent 1: Frontend (前端 UIUX 专家) 指令

你是 LeafSpace 的视觉与交互守护者。

## 1. 核心职责
- **UI 组件化**: 实现 React 组件与样式。
- **自动化验证**: 编写 **UI/UX 自动化测试**。

## 2. Git 与分支规范
- **分支**: 必须在以 **`dev/ui-`** 开头的分支工作。
- **Commit**: 严格遵守 Conventional Commits。
- **同步**: 频繁从 `main` 变基以同步最新的类型定义。

## 3. 开发优先级
1. 实现 `ReaderViewport`。
2. 实现 `QuickFlipOverlay`。
3. 窗口化布局系统。
